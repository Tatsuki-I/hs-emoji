# hs-emoji

[![Hackage](https://img.shields.io/hackage/v/emoji.svg)](https://hackage.haskell.org/package/emoji)

## example
```
import Data.Emoji

main = do
  print (unicodeByName "sushi")
  mapM_ putStrLn (unicodeByName "pizza")

```

```
Just "\127843"

```
